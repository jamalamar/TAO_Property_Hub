# **App Name**: TAO Property Hub

## Core Features:

- Asset Navigation: Implement a top-level navigation distinguishing between 'Total Assets' and 'Assets for Rent'.
- Data Organization: Organize property data by city (Acapulco, Cancun, Puerto Escondido, Ciudad Del Carmen) and property type (Commercial, Apartments, Houses, Land, Construction).
- Property Instance Display: Display specific property instances within each city and property type category (e.g., Acapulco > Commercial > Tienda Centro).
- Admin Modules: Provide access to three administrative modules: Legal, Administration, and Maintenance, with specified sub-sections.
- User Roles: Implement role-based access control to display either 'Socio Director' or 'Invitado (Guest)' views.
- AI Document Summarizer: Utilize an AI-powered tool that summarizes key details from uploaded documents within Legal and Administration modules, and generates concise reports, when appropriate.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) - A clean, vibrant blue (#4682B4) evokes trust and professionalism, suitable for a property management system in Mexico.
- Background color: HSL(210, 10%, 98%) - A very light blue (#F0F8FF) background provides a clean and modern backdrop in the style of properties in Mexico.
- Accent color: HSL(180, 50%, 40%) - A teal color (#40E0D0), which is complementary to blue, highlights key actions and provides a fresh feel.
- Body font: 'Inter', a grotesque-style sans-serif, provides a modern and neutral feel for both headlines and body text, ideal for the Mexican market.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean and modern icons to represent different property types, administrative modules, and actions, tailored for a Mexican audience.
- Implement a clear and intuitive layout that allows users to easily navigate between different cities, property types, and administrative modules, optimized for Spanish-speaking users in Mexico.
